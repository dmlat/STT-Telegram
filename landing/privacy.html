Политика конфиденциальности

Дата последнего обновления: 23 ноября 2025 г.


1. Сбор данных

Наш сервис (Telegram бот Voice2Text) обрабатывает аудиофайлы и голосовые сообщения, которые вы отправляете, исключительно с целью их транскрибации (перевода в текст). Мы не храним ваши аудиофайлы после завершения обработки.


2. Использование данных

Полученные данные используются только для предоставления услуги распознавания речи. Мы не передаем ваши данные третьим лицам, за исключением провайдеров API (OpenAI), необходимых для работы технологии распознавания.


3. Безопасность

Мы принимаем все необходимые меры для защиты ваших данных от несанкционированного доступа.


4. Контакты

Если у вас есть вопросы по поводу политики конфиденциальности, свяжитесь с нами через поддержку бота.
